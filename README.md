# Pavlov
Pet Info tracker for iRO:Odin (Restart)

[Download link](https://github.com/miatribe/Pavlov/releases/download/0.2/Pavlov.zip)

#### Must be ran as admin.

Arrow at bottom right can be used to cycle between RO clients
